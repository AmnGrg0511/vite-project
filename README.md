# vite-project

## Steps to run:
1. Clone the repo : 
    ```bash
    git clone git@github.com:AmnGrg0511/vite-project.git
    ```

2. Install the dependencies :
    ```bash
    npm install
    ```

3. Build the project :
    ```bash
    npm run build
    ```

4. Preview the project :
    ```bash
    npm run previe
    ```
