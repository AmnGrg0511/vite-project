# vite-project

## Steps to run:
1. Clone the repo: 
    ```bash
    git clone git@github.com:AmnGrg0511/vite-project.git
    ```

1. Change current working directory:
    ```bash
    cd vite-project
    ```

1. Install the dependencies:
    ```bash
    npm install
    ```

1. Build the project:
    ```bash
    npm run build
    ```

1. Preview the project:
    ```bash
    npm run previe
    ```
